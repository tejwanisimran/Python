# Selection in Python 
  Selection refers to ability of program to execute certain blocks of the code based on specific conditions.

  # 1. if-statement : 
  
Syntax : 
    
          if condition :  
                  # block of code(executed if the condition is true)

  # 2. if-else-statement : 
  
  Syntax : 
  
            if condition :  
                  # block A  
            else :  
                  # block B  

  # 3. if-elif-else-ladder :  
  We use this when we need to evaluate multiple conditions  
  
  Syntax :   
  
              if condition1 :  
                    # block A  
              elif condition2 :  
                    # block B  
              elif condition3 :   
                    # block C  
              else :   
                    # block D  
  # 4. Nested if-statements :   
  
 An if statement inside another if statement.  

# Iteration in Python   

  # 1. for-loop : 
  
  Syntax :   
  
              for variable in sequence :  
                    # block of code  

  # 2. while-loop :  
  
  Syntax :  
  
              while condition :  
                  # block of code  

# Lambda Function in Python 

A lambda function python is a small anonymous function in python   
It can have only one expression and any number of arguments  

Syntax : 

          labda arguments : expression
