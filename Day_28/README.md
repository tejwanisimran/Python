### 🔹 Loss Functions
- Mean Squared Error (MSE)
- (13_LossFunctions_MSE)
```
Calculates average squared difference between actual and predicted values
```
- Mean Absolute Error (MAE)
- (14_LossFunctions_MAE)
```
Calculates average absolute difference between actual and predicted values
```
- Binary Cross Entropy (BCE)
- (15_LossFunctions_Binary_Cross_Entropy)
```
Used for binary classification problems
Includes protection against log(0) errors
```
