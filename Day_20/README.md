### 4. Model Evaluation using R² Score
- Understanding R-squared (coefficient of determination)
- Evaluating model performance:
### ✔ Perfect Prediction
- R² = 1 (ideal model)
### ✔ Poor Prediction
- Negative R² (very bad model)
### ✔ Moderate Prediction
- R² between 0 and 1.
