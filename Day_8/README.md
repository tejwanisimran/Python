# Object oriented programming in Python :   

- Python is multi-programming language.
- Means python supports Object oriented programming approach.
  # To do programming in Object orientation we need to know these concepts :

- What is class , how to design a class , what are characteristics , behaviours , how to write a constructor , destructor,etc
  # 1. class :
- It is considered as the blueprint of the object.  
- Ans class contains 2 thisngs as characteristics & behaviours.  
  # 2. Characteristics :
- Characteristics are variables of the class on which can perform operations.
- When we create object of that class memory for the characteristics gets allocated.  
  # 3. Behaviours :
- Behaviours means the methods which performs operations on characteristics.
- By using the object we can call the methods.
  # Constructor :
- Construtor is the soecial function which gets called when the object is created.
- Syntax :
  
        def __init__(self):
  # Destructor :
- Destructor is the special function which gets called when the object is destroyed or goes out of scope.
- Syntax :  
-         def __del__(self):
