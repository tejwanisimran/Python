## Day 14 : 

## Conceptual codes to understand Data Backup & Automation 
---
### Learnt concepts like : 
---
### 1. File Backup Automation
- Learned how to automate file copying from source to destination using shutil.
- This helps in creating backups without manual effort and ensures data safety.
---
### 2. Incremental Backup
- Implemented logic to copy only new or modified files instead of all files.
- This improves performance and avoids unnecessary duplication.
---
### 3. Checksum (MD5 Hashing)
- Used MD5 hashing to compare file contents instead of file names.
- This ensures accurate detection of file changes and duplicates.
---
### 4. File Synchronization
- Learned how to keep the destination folder updated with the source folder.
- Ensures both locations have the latest version of files.
---
### 5. Zip Archive Creation
- Used zipfile module to compress backup data into .zip files.
- This helps in saving storage space and organizing backups.
---
### 6. Task Scheduling (schedule)
- Automated backup execution at fixed time intervals.
- Reduces manual work and enables continuous data protection.
