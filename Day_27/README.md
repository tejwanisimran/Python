## Day 27 : 
## 🧠 Artificial Neural Network (ANN)
---
### 📂 Files Included : 
---
### 🔹 1. Basic Neuron
- 1_ANN_Single_Neuron.py  
  Implements a single artificial neuron using inputs, weights, and bias.
  Uses dot product and ReLU activation.
---
### 🔹 2. Activation Functions
- 2_Activation_Relu.py  
  Demonstrates ReLU activation and neuron forward pass with visualization.
- 3_Activation_Sigmoid.py  
  Implements Sigmoid activation for probability-based output.
- 4_Activation_Sigmoid_vs_Relu.py  
  Compares ReLU and Sigmoid functions graphically.
---
### 🔹 3. Neural Network Structure
- 5_Layered_Artificial_Neural_Network.py  
  Implements a full ANN:  
  Input Layer  
  Hidden Layer (ReLU)  
  Output Layer (Sigmoid)  
- 6_Layered_Artificial_Neural_Network_Simplified.py  
  Step-by-step forward propagation with detailed explanation.
- 7_Layered_Artificial_Neural_Network_Graphical.py  
  Visual representation of ANN structure and outputs using graphs
---
### 🔹 4. Training Process
- 8_ANN_Training_Steps.py  
  Demonstrates how weights are updated using error and learning rate.
- 9_ANN_Training_Steps_Graphical.py  
  Shows training with graphs:
  ```
  Loss reduction
  Prediction improvement
  ```
---
### 🔹 5. Multi-Neuron ANN Training
- 10_ANN_MultiNeuron_Training_Steps.py  
  Multi-neuron ANN with manual weight updates.
- 11_ANN_MultiNeuron_Training_Steps_Graphical.py  
  Visualization of training progress and weights.
- 12_ANN_MultiNeuron_Training_Steps_Graphical_Animation.py  
  Animated training visualization of ANN learning process.
---
### ⚙️ Concepts Covered
- Neuron structure (Inputs, Weights, Bias)  
- Weighted sum calculation  
- Activation functions:  
- ReLU  
- Sigmoid  
- Forward propagation  
- Loss calculation (Squared Error)  
- Gradient-based weight updates (simplified)  
- Multi-layer neural networks  
- Visualization of learning process
