## Command Line Arguments : 

- Command line arguments are values passed to a program when you run it from the terminal (command prompt).
- They allow you to give input to your program without hardcoding values.
- Syntax :
  
```
import sys

# Access command line arguments
sys.argv

```

--- 

## Schedular in python : 

- A scheduler in Python is used to run tasks automatically at a specific time or after a certain interval—instead of
  manually executing them.
